# Источники данных

## Основной источник: Әділет (adilet.zan.kz)

### Описание
Информационно-правовая система нормативных правовых актов Республики Казахстан. Администрируется Институтом законодательства и правовой информации при Министерстве юстиции РК.

### Содержание
- **230 000+** нормативных правовых актов
- Документы **с 1947 по 2026 год**
- Три языка: казахский, русский, английский

### URL-паттерны

```
# Документ (полный текст)
https://adilet.zan.kz/rus/docs/{DOC_ID}

# Метаданные
https://adilet.zan.kz/rus/docs/{DOC_ID}/info

# История изменений
https://adilet.zan.kz/rus/docs/{DOC_ID}/history

# Перекрёстные ссылки
https://adilet.zan.kz/rus/docs/{DOC_ID}/links

# Скачать PDF
https://adilet.zan.kz/rus/docs/{DOC_ID}/download

# Скачать DOCX
https://adilet.zan.kz/rus/docs/{DOC_ID}/download/docx

# Архивная версия на дату
https://adilet.zan.kz/rus/archive/docs/{DOC_ID}/{DD.MM.YYYY}
```

### Формат DOC_ID
- Позиция 1: тип документа (`K`=Кодекс, `Z`=Закон, `P`=Постановление, `U`=Указ, `V`=Приказ)
- Позиции 2-3: год (двузначный)
- Позиции 4-10/11: номер документа с нулями

Примеры:
- `K950001000_` — Конституция РК (1995, №1000)
- `K1500000414` — Трудовой кодекс (2015, №414)
- `Z1600000480` — Закон о правовых актах (2016, №480)

### Метаданные документа (страница /info)
- Дата принятия
- Дата последнего изменения
- Форма акта (тип документа)
- Юридическая сила
- Орган, принявший акт
- Сфера правоотношений
- Регион действия
- Статус: действующий (`status_upd`) / утративший силу (`status_yts`)

### Ограничения
- **Нет публичного API** — только HTML-парсинг
- robots.txt разрешает парсинг `/*/docs/`, запрещает `/*/search/` и `/*/archive/`
- Rate limiting: ~2 запроса/сек для уважительного парсинга

## Демо-набор данных

Для демонстрации выбран кластер **трудового права** (~50 документов):

### Приоритет 1: Ядро (обязательно)
| Документ | ID | Причина выбора |
|----------|-----|---------------|
| Конституция РК | K950001000_ | Фундамент, ссылки отовсюду |
| Трудовой кодекс | K1500000414 | Основной фокус, 400+ статей |
| Предпринимательский кодекс | K1500000375 | Пересечение с трудовым правом |
| Гражданский кодекс (общая часть) | K940001000_ | Контракты, ответственность |
| Гражданский кодекс (особенная часть) | K990000409_ | Смежные договоры |
| Налоговый кодекс | K1700000120 | Налоговые обязанности работодателя |

- Подзаконные акты по трудовым отношениям
